# Safe Prompt Studio

Safe Prompt Studio 是一套「明確成年人、非露骨」的時尚、泳裝、健身、瑜珈與生活寫真提示詞建構工具。

目前正在進行 v0.1 Phase 1，已完成 Vite + React + TypeScript 骨架、型別、詞庫 schema、核心 Prompt Builder 與最小可操作介面。

## 本機開發

```bash
npm install
npm run dev
npm run build
npm run preview
```

完整文件、收藏功能與 GitHub Pages 工作流程將於後續階段補齊。
